# Area Platform

AI-powered customer support platform with Google Sign-In, console dashboard, full documentation, and more.

## Setup
1. Clone repo
2. Run `firebase login`
3. Run `firebase init hosting` (select public folder)
4. Run `firebase deploy`

## Features
- Firebase Google Auth
- Protected Console with API keys, usage, models
- Full Documentation
- Blog, Pricing, About, Contact pages
- Butterfly logo and modern UI
